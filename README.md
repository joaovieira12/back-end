# Sistema Escolar - Back-end

Back-end do Projeto Final FullStack (Node.js + Express + PostgreSQL + JWT).

> Este README vai sendo completado a cada parte do back-end.

## Estrutura de pastas

```
backend/
├── config/         -> conexão com o banco
├── models/         -> queries SQL de cada entidade
├── controllers/    -> regras de negócio das rotas
├── routes/         -> definição das rotas Express
├── middlewares/    -> autenticação, roles e validação
├── db/             -> schema.sql com as tabelas
├── app.js          -> configuração do Express (middlewares globais, rotas)
└── server.js        -> ponto de entrada (inicia o servidor)
```
